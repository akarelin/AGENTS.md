# manage

Admin and management tools with 3 sub-skills.

## Installation

```
/plugin install manage@akarelin-skills
```

## Sub-skills

### manage-sessions
Manage Claude Code session folders across repos and hosts. Sync, list, resume, rename, and cleanup operations.

### manage-skills
End-to-end workflow for improving plugin skills: gather feedback, patch code, test, rebuild `.plugin` files, and deploy.

### manage-m365
Microsoft 365 tenant administration via Graph beta API. User CRUD, group management, Teams admin, license assignment, directory roles, audit logs, devices, security alerts/score. Admin-only. Requires `pip install msal requests`.
