# Skill Manager

Manage Cowork plugin skills end-to-end: gather feedback, patch code, test, rebuild, and deploy `.plugin` files.

## Commands

- `/patch-skill [plugin-name]` — Fix issues in a plugin and redeploy it.

## Skills

- **skill-manager** — Full workflow reference for plugin maintenance. Triggered by "fix skill", "patch plugin", "update skill", "deploy plugin", etc.
